[Turbopack](https://turbo.build/pack)(ベータ版)は、JavaScriptやTypeScriptに最適化されたインクリメンタルバンドラで、Rustで書かれており、Next.jsに組み込まれています。

## Usage

Turbopackは、Next.jsの`pages`ディレクトリーと`app`ディレクトリーの両方で使用することができ、より速いローカルのdevelopmentを可能にします。Turbopackを有効にするには、Next.js development serverを実行する際に`--turbo`フラグを使用してください。

```json:package.json
{
  "scripts": {
    "dev": "next dev --turbo",
    "build": "next build",
    "start": "next start",
    "lint": "next lint"
  }
}
```

## Supported Features

Turbopackの現在サポートされている機能の詳細については、[documentation](https://turbo.build/pack/docs/features)をご覧ください。

## Unsupported Features

TTurbopackは現在、`next dev`のみをサポートしており、`next build`はサポートしていません。安定性に向けて前進するにつれて、buildのサポートに取り組んでいます。
