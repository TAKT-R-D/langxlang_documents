Next.js は、あなたのアプリケーションをスタイリングするためのさまざまな方法をサポートしています。これには次のものが含まれます:

- **グローバル CSS**: 使いやすさと、従来の CSS に慣れている人には馴染みがありますが、アプリケーションが拡大するにつれて CSS バンドルが大きくなる可能性があり、また styles の管理が難しくなる可能性があります。
- **CSS Modules**: ネーミングコンフリクトを回避し、保守性を向上させるために、ローカルスコープの CSS クラスを作成します。
- **Tailwind CSS**: ユーティリティクラスを組み合わせて迅速にカスタムデザインを可能にする、ユーティリティファーストの CSS フレームワーク。
- **Sass**: 変数、ネストされたルール、ミックスインなどの機能で CSS を拡張する、人気のある CSS プリプロセッサ。
- **CSS-in-JS**：あなたの JavaScript 　 component に直接 CSS を埋め込むことで、 dynamic そしてスコープされたスタイリングを可能にします。
