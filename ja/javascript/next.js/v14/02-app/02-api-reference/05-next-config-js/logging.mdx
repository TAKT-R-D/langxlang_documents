あなたはロギングレベルを設定することができ、development モードで Next.js を実行する際に全ての URL がコンソールに記録されるかどうかを設定することができます。

現在、`logging`は、`fetch`の API を使ってデータを取得する際にのみ適用されます。まだ Next.js 内の他のログには適用されていません。

```js:next.config.js
module.exports = {
  logging: {
    fetches: {
      fullUrl: true,
    },
  },
}
```
