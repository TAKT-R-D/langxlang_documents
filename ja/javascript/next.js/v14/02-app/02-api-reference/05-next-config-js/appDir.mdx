> **Good to know**: このオプションは、**もう不要**となりました。これは、 Next.js 13.4より。 App Router は現在、安定しています。

 App Router ([`app`ディレクトリ](/ja/javascript/next.js/v14/02-app/01-building-your-application/01-routing))は、[レイアウト](/ja/javascript/next.js/v14/02-app/01-building-your-application/01-routing/02-pages-and-layouts)、[Server Components](/ja/javascript/next.js/v14/02-app/01-building-your-application/03-rendering/01-server-components)、[ストリーミング](/ja/javascript/next.js/v14/02-app/01-building-your-application/01-routing/04-loading-ui-and-streaming)、および[コロケーションデータ取得](/ja/javascript/next.js/v14/02-app/01-building-your-application/02-data-fetching)のサポートを可能にします。

U`app`ディレクトリを使用すると、自動的に[React Strict Mode](https://react.dev/reference/react/StrictMode)が有効になります。[徐々に`app`を採用する方法](/ja/javascript/next.js/v14/02-app/01-building-your-application/11-upgrading/02-app-router-migration#migrating-from-pages-to-app)について学びましょう。
