Next.js は、 default で各ページに[etags](https://en.wikipedia.org/wiki/HTTP_ETag)を生成します。あなたの cache strategy によっては、 HTML ページの etag 生成を無効にしたいかもしれません。

`next.config.js`を開き、`generateEtags`オプションを無効にします：

```js:next.config.js
module.exports = {
  generateEtags: false,
}
```
