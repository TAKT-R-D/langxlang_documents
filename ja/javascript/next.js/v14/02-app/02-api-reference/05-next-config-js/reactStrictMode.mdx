> **Good to know**: Next.js 13.4 以降、Strict Mode は`app` router で default により`true`になりますので、上記の設定は`pages`専用です。 依然として Strict Mode を無効にすることができます。 `reactStrictMode: false`に設定します。

> **推奨**: あなたの Next.js アプリケーションの未来をよりよく準備するために、強く Strict Mode を有効にすることをおすすめします。

React の[Strict Mode](https://react.dev/reference/react/StrictMode)は、アプリケーションの潜在的な問題を強調するための development モードだけの機能です。これは、安全でないライフサイクル、レガシーな API の使用、そして他の number の機能を識別するのに役立ちます。

Next.js runtime は Strict Mode に準拠しています。Strict Mode を有効にするには、あなたの`next.config.js`で次のオプションを設定してください：

```js:next.config.js
module.exports = {
  reactStrictMode: true,
}
```

もしあなた自身やあなたのチームが、アプリケーション全体で Strict Mode を使用する準備ができていない場合でも、問題ありません！`<React.StrictMode>`を使用してページごとに段階的に移行することができます。
