> **Good to know**: Next.js 13.4以降、Strict Modeは`app` router で default により`true`になりますので、上記の設定は`pages`専用です。 依然としてStrict Modeを無効にすることができます。 `reactStrictMode: false`に設定します。

> **推奨**: あなたのNext.jsアプリケーションの未来をよりよく準備するために、強くStrict Modeを有効にすることをおすすめします。

Reactの[Strict Mode](https://react.dev/reference/react/StrictMode)は、アプリケーションの潜在的な問題を強調するためのdevelopmentモードだけの機能です。これは、安全でないライフサイクル、レガシーなAPIの使用、そして他のnumberの機能を識別するのに役立ちます。

Next.js runtimeはStrict Modeに準拠しています。Strict Modeを有効にするには、あなたの`next.config.js`で次のオプションを設定してください：

```js:next.config.js
module.exports = {
  reactStrictMode: true,
}
```

Iもしあなた自身やあなたのチームが、アプリケーション全体でStrict Modeを使用する準備ができていない場合でも、問題ありません！`<React.StrictMode>`を使用してページごとに段階的に移行することができます。
