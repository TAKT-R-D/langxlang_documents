[静的に型付けされたリンク](/ja/javascript/next.js/v14/02-app/01-building-your-application/07-configuring/01-typescript#statically-typed-links)の実験的なサポート。この機能を使用するには、プロジェクトで App Router と TypeScript を使用する必要があります。

```js:next.config.js
/** @type {import('next').NextConfig} */
const nextConfig = {
  experimental: {
    typedRoutes: true,
  },
}

module.exports = nextConfig
```
