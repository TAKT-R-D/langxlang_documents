実験的な `instrumentationHook` オプションは、 Next.js App 内の[`instrumentation` ファイル](/ja/javascript/next.js/v14/02-app/02-api-reference/02-file-conventions/instrumentation)を介して instrumentation を設定することを可能にします。

```js:next.config.js
module.exports = {
  experimental: {
    instrumentationHook: true,
  },
}
```
