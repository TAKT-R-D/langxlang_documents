Next.jsは、ローカルパッケージ(モノレポなど)または外部依存関係(`node_modules`)から自動的にトランスパイルやバンドル化を行うことができます。これにより、`next-transpile-modules`パッケージが置き換えられます。

```js:next.config.js
/** @type {import('next').NextConfig} */
const nextConfig = {
  transpilePackages: ['@acme/ui', 'lodash-es'],
}

module.exports = nextConfig
```
