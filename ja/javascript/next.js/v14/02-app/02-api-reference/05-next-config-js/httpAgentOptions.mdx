18より前のNode.jsVersionでは、Next.jsが自動的に`fetch()`を[undici](/ja/javascript/next.js/v14/04-architecture/supported-browsers#polyfills)でpolyfillsし、[HTTP Keep-Alive](https://developer.mozilla.org/docs/Web/HTTP/Headers/Keep-Alive)をdefaultで有効にします。

すべての `fetch()` 呼び出しのための HTTP Keep-Aliveを server-sideで無効にするには、`next.config.js`を開き、`httpAgentOptions` configを追加します：

```js:next.config.js
module.exports = {
  httpAgentOptions: {
    keepAlive: false,
  },
}
```
