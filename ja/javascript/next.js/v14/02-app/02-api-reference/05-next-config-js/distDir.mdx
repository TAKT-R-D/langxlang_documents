あなたは、代わりに`.next`を使用するのではなく、カスタムの build ディレクトリに使用する名前を指定することができます。

`next.config.js`を開いて、`distDir` config を追加してください：

```js:next.config.js
module.exports = {
  distDir: 'build',
}
```

もし今、`next build`を実行すると、Next.js は`build`を使い、default の `.next` フォルダの代わりになります。

> `distDir`はプロジェクトディレクトリから出ては**いけません**。例えば、`../build`は**無効な**ディレクトリです。
