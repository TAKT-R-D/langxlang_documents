このドキュメントのセクションでは、**Metadata ファイルの規約**について説明します。ファイルベースの metadata は、特別な metadata ファイルを route セグメントに追加することで定義できます。

それぞれのファイル規約は、静的なファイル(例：`opengraph-image.jpg`)を使用して定義することができます、または dynamic なバリアントを使用し、code でファイルを生成する(例：`opengraph-image.js`)。

ファイルが定義されると、Next.js は自動的にファイルを配信し(production にはキャッシュのためのハッシュ付き)、関連する head 要素を正しい metadata(例えば、アセットの URL、ファイルの type、そして image size)で更新します。
