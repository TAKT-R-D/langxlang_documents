`draftMode`関数を使用すると、[Server Component](/ja/javascript/next.js/v14/02-app/01-building-your-application/03-rendering/01-server-components)内の[Draft Mode](/ja/javascript/next.js/v14/02-app/01-building-your-application/07-configuring/11-draft-mode)を検出することができます。

```jsx:app/page.js
import { draftMode } from 'next/headers'

export default function Page() {
  const { isEnabled } = draftMode()
  return (
    <main>
      <h1>My Blog Post</h1>
      <p>Draft Mode is currently {isEnabled ? 'Enabled' : 'Disabled'}</p>
    </main>
  )
}
```
