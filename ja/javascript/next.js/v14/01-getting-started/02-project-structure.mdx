このページでは、Next.jsアプリケーションのプロジェクト構造の概要を提供しています。最上位のファイルやフォルダ、設定ファイル、そして `app` および `pages` ディレクトリ内のルーティング規則について説明します。

ファイルやフォルダの名前をクリックして、各規約について詳しく学びましょう。

## Top-level files

トップレベルのファイルは、アプリケーションの設定、依存関係の管理、middlewareの実行、監視ツールの統合、およびenvironment variablesの定義に使用されます。

| | |
| ------------------------------------------------------------------------------------------- | --------------------------------------- |
| **Next.js** | |
| [`next.config.js`](/ja/javascript/next.js/v14/02-app/02-api-reference/05-next-config-js) | Next.js の設定ファイル |
| [`package.json`](/ja/javascript/next.js/v14/01-getting-started/01-installation#manual-installation) | プロジェクトの依存関係と scripts |
| [`instrumentation.ts`](/ja/javascript/next.js/v14/02-app/01-building-your-application/06-optimizing/09-instrumentation) | OpenTelemetryと Instrumentation ファイル |
| [`middleware.ts`](/ja/javascript/next.js/v14/02-app/01-building-your-application/01-routing/13-middleware) | Next.js request middleware |
| [`.env`](/ja/javascript/next.js/v14/02-app/01-building-your-application/07-configuring/03-environment-variables) | Environment variables |
| [`.env.local`](/ja/javascript/next.js/v14/02-app/01-building-your-application/07-configuring/03-environment-variables) | ローカルの environment variables |
| [`.env.production`](/ja/javascript/next.js/v14/02-app/01-building-your-application/07-configuring/03-environment-variables) | Production environment variables |
| [`.env.development`](/ja/javascript/next.js/v14/02-app/01-building-your-application/07-configuring/03-environment-variables) | Development environment variables |
| [`.eslintrc.json`](/ja/javascript/next.js/v14/02-app/01-building-your-application/07-configuring/02-eslint) | ESLint の設定ファイル |
| `.gitignore` | 無視するGitファイルやフォルダ |
| `next-env.d.ts` | Next.js のための TypeScript declaration ファイル |
| `tsconfig.json` | TypeScript の設定ファイル |
| `jsconfig.json` | JavaScript の設定ファイル |
