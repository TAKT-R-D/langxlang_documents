Next.js 13 以前では、Pages Router は Next.js で routes を作成する主要な方法でした。これは直感的なファイルシステムの router を使用して、各ファイルを routes にマッピングしました。 Pages Router は、新しい versions の Next.js でもサポートされていますが、React の最新の機能を活用するために新しい[App Router](/ja/javascript/next.js/v14/02-app)に移行することをお勧めします。

このドキュメンテーションのセクションは、Pages Router を使用する既存のアプリケーション用です。
