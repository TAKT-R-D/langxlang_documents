Next.js のデータ取得では、アプリケーションの使用事例に応じて、異なる方法でコンテンツを render できます。これには、**Server-side Rendering**または**Static Generation**を使用した事前レンダリング、**Incremental Static Regeneration**を使用して runtime でコンテンツを更新または作成することが含まれます。
