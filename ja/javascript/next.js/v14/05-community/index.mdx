週に500万回以上のダウンロードを持つ、Next.jsは、world全体に広がる大規模かつ活動的な開発者コミュニティを擁しています。以下は、私たちのコミュニティに参加する方法です：

## Contributing

あなたがNext.jsのdevelopmentに貢献できる方法はいくつかあります：

- [Documentation](/ja/javascript/next.js/v14/05-community/01-contribution-guide): 私たちのユーザーがNext.jsの使い方を理解するのを助けるために、改善点を提案したり、新しいセクションを書いたりしてください。
- [事例](https://github.com/vercel/next.js/tree/canary/contributing/examples)：新しい事例を作成したり、既存のものを改善したりして、開発者がNext.jsを他のツールやサービスと統合するのを支援します。
- [Codebase](https://github.com/vercel/next.js/tree/canary/contributing/core): 基盤となるアーキテクチャについて詳しく学び、バグ修正、error、新機能の提案に貢献してください。

## Discussions

もしあなたがNext.jsについての質問がある、または他の人を助けたい場合は、いつでも会話に参加することを歓迎します：

- [GitHub ディスカッション](https://github.com/vercel/next.js/discussions)
- [Discord](https://discord.com/invite/bUG2bvbtHy)
- [Reddit](https://www.reddit.com/r/nextjs)

## Social Media

最新のアップデートについては、[Twitter](https://twitter.com/nextjs)で Next.js をフォローし、Next.jsのビデオを視聴するためには[Vercel YouTubeチャンネル](https://www.youtube.com/@VercelHQ)を購読してください。

## Code of Conduct

W私たちは、包括的で歓迎的なコミュニティを作ることを信じています。そのため、すべてのメンバーに私たちの[Code of Conduct](https://github.com/vercel/next.js/blob/canary/CODE_OF_CONDUCT.md)を遵守するよう求めます。このdocumentは、参加者の行動に対する私たちの期待を概説しています。それを読んで、安全で尊重のある環境を維持するのを助けてください。
