## Introduction

[Laravel Mix](https://github.com/laravel-mix/laravel-mix)は、[Laracasts](https://laracasts.com)の作成者 Jeffrey Way によって development されたパッケージで、いくつかの一般的な CSS と JavaScript のプリプロセッサを使用して、Laravel アプリケーションのための[webpack](https://webpack.js.org)の build 手順を定義するための流暢な API を提供します。

つまり、Mix を使えば、あなたのアプリケーションの CSS や JavaScript ファイルを簡単にコンパイルしてミニファイさせることができます。シンプルな method チェーンを通じて、あなたのアセットの pipeline を流暢に定義することが可能です。例えば：

```js
mix
  .js("resources/js/app.js", "public/js")
  .postCss("resources/css/app.css", "public/css");
```

webpack とアセットコンパイルの開始について混乱したり、圧倒されたことがある場合、あなたは Laravel Mix を気に入るでしょう。しかし、あなたが application を development する際にそれを使うことは必須ではありません。あなたは自由に好きなアセット pipeline ツールを使うか、或いは全くないことも選択できます。
